# Syne Admin Application 

#### Run locally on your machine

You need to have installed Node, NPM and Gulp.

## Installation

## Clone the repo

```bash
$ https://github.com/synegit/SyneApps.git
```

## cd in the repo

```bash
$ cd SyneApps
```

## install dependencies

```bash
$ npm install
```

## install gulp globally
```bash
$ npm install gulp-cli -g
```

## run application

```bash
$ npm start
```

## build application

```bash
$ npm build
```
